# KVM (Kernel-based Virtual Machine)

KVM 是 Linux 核心內建的虛擬化技術，提供完整的虛擬化解決方案。

## 安裝指南

- [CentOS KVM 安裝](centos-kvm/) - 在 CentOS 上建置 KVM 虛擬化環境

## 主要元件

- **KVM**: 核心虛擬化模組
- **QEMU**: 硬體模擬器
- **libvirt**: 虛擬化管理 API
- **virt-manager**: 圖形化管理工具

## 管理工具

- `virsh`: 命令列管理工具
- `virt-install`: 虛擬機安裝工具
- `virt-viewer`: 虛擬機控制台
- `virt-manager`: GUI 管理介面
