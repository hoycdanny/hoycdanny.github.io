# PXE 網路啟動伺服器

PXE (Preboot Execution Environment) 網路啟動伺服器安裝腳本。

## 📋 說明

PXE 允許電腦透過網路啟動作業系統，常用於：
- 大規模系統部署
- 無磁碟工作站
- 系統救援和維護
- 自動化安裝

## 🚀 安裝

```bash
chmod +x install.sh
sudo ./install.sh
```

## 📖 功能

- DHCP 伺服器配置
- TFTP 伺服器設定
- PXE 啟動檔案配置
- 網路安裝環境

## 🎯 使用場景

- 資料中心批次部署
- 實驗室環境管理
- 系統自動化安裝
- 無磁碟終端機
