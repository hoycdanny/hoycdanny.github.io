# Container Storage 容器儲存

容器環境的持久化儲存解決方案。

## 內容

- [HPE Nimble Storage](nimble/) - Nimble Storage 與 Docker/Kubernetes 整合指南

## 儲存方案比較

容器環境常見的儲存方案：
- **本地儲存**: 簡單但不適合生產環境
- **NFS**: 傳統網路儲存，易於設定
- **iSCSI**: 區塊級儲存，效能較好
- **CSI 驅動**: 現代化的容器儲存介面
