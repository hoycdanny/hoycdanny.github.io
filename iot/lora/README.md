# LoRa Docker Container

LoRa (Long Range) 物聯網通訊協定 Docker 容器。

## 📋 說明

LoRa 是一種低功耗廣域網路 (LPWAN) 通訊技術，適用於物聯網設備的長距離通訊。

## 🚀 使用

```bash
# 建置映像
docker build -t lora:latest .

# 執行容器
docker run -d lora:latest
```

## 🎯 特性

- **長距離通訊**: 可達數公里
- **低功耗**: 適合電池供電設備
- **穿透力強**: 適合城市環境
- **低成本**: 硬體成本低

## 📖 應用場景

- 智慧城市感測器
- 農業監控
- 資產追蹤
- 環境監測
- 智慧建築
