# HPE Ezmeral

HPE Ezmeral 是企業級的資料與分析平台。

## 內容

- [Data Fabric](data_fabric/) - HPE Ezmeral Data Fabric 管理與開發指南

## HPE Ezmeral Data Fabric 特色

- 全球命名空間
- 多租戶支援
- 企業級安全性
- 高可用性與災難復原
- 與 Hadoop 生態系統相容

## 主要功能

- **分散式檔案系統**: 類似 HDFS 但效能更好
- **NoSQL 資料庫**: MapR-DB (JSON 與 Binary)
- **串流處理**: MapR Streams (相容 Kafka API)
- **物件儲存**: S3 相容的物件儲存介面
